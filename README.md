## вңҰ Apa itu Androbot?

**Androbot** adalah WhatsApp bot framework ringan berbasis [Baileys](https://github.com/WhiskeySockets/Baileys), hasil fork dari [raganork-md](https://github.com/souravkl11/raganork-md) вҖ” dioptimalkan agar bisa berjalan langsung di **Android via Termux**, tanpa perlu VPS maupun root.

> Satu HP lama pun bisa jadi server bot WhatsApp kamu 24/7.

## вңҰ Instalasi di Android (Termux)

> Cara paling mudah menjalankan Androbot вҖ” langsung dari HP kamu.

### Langkah 0 вҖ” Persiapan

- Install **Termux** dari [F-Droid](https://f-droid.org/packages/com.termux/) *(bukan Play Store)*
- Session bersifat **opsional** (bot bisa login via QR/pairing langsung)
- Jalankan sekali: `termux-setup-storage`

---

### Langkah 1 вҖ” Jalankan Script Installer

```bash
chmod +x deploy-androbot.sh
./deploy-androbot.sh
```

Script **`deploy-androbot.sh`** akan otomatis mengurus semuanya:

```
вң“  Update & install dependency (git, nodejs-lts, ffmpeg, yarn, pm2)
вң“  Clone repo ke ~/androbot
вң“  Tanya Session (opsional), nama bot, nomor SUDO, timezone
вң“  Buat file .env otomatis
вң“  Jalankan bot via PM2
```

---

### Langkah 2 вҖ” Cegah Bot Mati di Background

Android suka mematikan proses background. Lakukan ini agar bot tetap hidup:

```bash
# Aktifkan wake lock
termux-wake-lock
```

Lalu di pengaturan Android:
> **Pengaturan вҶ’ Aplikasi вҶ’ Termux вҶ’ Baterai вҶ’ Tidak Dioptimalkan**

Untuk autostart PM2 saat Termux dibuka ulang:

```bash
pm2 startup
pm2 save
```

## вңҰ Update Bot

Kapanpun repo GitHub ada perubahan, jalankan:

```bash
bash ~/androbot/update-androbot.sh
```

Script **`update-androbot.sh`** akan:

```
вҶ’  Cek apakah ada commit baru dari GitHub
вҶ’  Jika ADA update  : yarn install + restart otomatis
вҶ’  Jika TIDAK ADA   : beri tahu kamu, tidak restart
```

Tidak ada update yang terlewat, tidak ada restart yang sia-sia.

## вңҰ Instalasi Manual (VPS / Linux)

```bash
# Install dependency global
npm install -g yarn pm2

# Clone & masuk folder
git clone https://github.com/celode/androbot
cd androbot

# Install dependency project
yarn install

# Buat .env (lihat bagian konfigurasi di bawah)
cp .env.example .env

# Jalankan
npm start
```

---

## вңҰ Konfigurasi `.env`

Buat file `.env` di root folder dengan isi berikut:


| Variable |
|---|---|
| SESSION=local | 
| USE_LOCAL_AUTH=true | 
| USE_SERVER=false | 
| MODE=public |
| HANDLERS=. |
| # opsional untuk pairing code: |
| # PAIRING_NUMBER=628xxxxxxxxxx |
---

## вңҰ Perintah PM2

```bash
pm2 logs androbot        # Lihat log secara live
pm2 status               # Cek status semua proses
pm2 restart androbot     # Restart bot
pm2 stop androbot        # Stop bot
```

## вңҰ Perintah Bot

Prefix default: `.`

| Perintah | Fungsi |
|---|---|
| `.list` | Tampilkan semua command |
| `.ping` | Cek response time |
| `.restart` | Restart bot *(sudo only)* |
| `.shutdown` | Matikan bot *(sudo only)* |
